# 🛒 Retail Management System

A Python-based retail management system to manage products, sales, stock, and employee records using CSV files.

---

## 🚀 Features

### 🔐 Admin Module
- Add, delete and update products
- Manage employee records

### 💰 Sales Module
- Generate customer bills
- Store sales data
- View sales reports

### 📦 Stock Module
- View stock availability
- Update stock after sales
- Stock analysis using graphs

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- CSV (for data storage)

---

## 📂 Project Files

- main.py → Main program
- product_list.csv → Product data
- quantity_list.csv → Stock data
- emp_list.csv → Employee records
- product_sold.csv → Sales data
- customer_detail.csv → Customer info
- password.txt → Passwords for modules
